# Improvements

## 현재 구현된 기능

- 바른손 리플렛 제작 신청 웹 폼 (Next.js 기반)
- Google Sheets 연동을 통한 신청 데이터 자동 저장 (Apps Script 웹앱 방식)
- Slack 웹훅을 통한 실시간 알림 구조 (웹훅 URL 설정 시 활성화)
- 신청 항목: 회사명, 담당자명, 연락처, 이메일, 주소, 제휴사 코드, 용도(샘플팩/리플렛), 수량, 납기일, 파일 첨부

## 향후 개선 사항

- 첨부 파일을 Google Drive에 자동 업로드하고 스프레드시트에 링크 연결
- 신청 완료 후 고객에게 자동 확인 이메일 발송 기능
- 신청 내역 조회 및 진행 상태 확인 페이지 추가
- 관리자 대시보드 구현 (신청 통계, 처리 현황 등)
- 모바일 반응형 UI 최적화 및 접근성 개선
